# pandoc-markdown-table-formatter

Pandoc Markdown table formatter for Vim.  

This vim plugin supports pandoc grid tables only.

## Commands

* :FormatPandocMDTable

## Example

+--------+----------+--------+
| left   | centered | right  |
+:=======+:========:+=======:+
| 左寄せ | 中央構え | 右寄せ |
+--------+----------+--------+
